§ 01

The strategy

Influencers, brand creators, and culinary launchers want a surface that feels editorial, aspirational, and Instagram-grade. Manufacturers, printers, and warehouses want clarity, data density, and zero ornamental friction. The system serves both — never compromised toward either pole.

CREATOR SURFACES

Editorial. Warm. Aspirational.

Comfortable spacing, generous imagery, occasional Fraunces moments for hero copy. Hover delight in marketplace cards. The platform should feel like it earns the right to host someone's brand launch.

  • Marketplace · Brand Assets · Design Studio canvas
  • Comfortable density (24–32px section spacing)
  • Fraunces serif for headers and editorial moments
  • Marketing site, niche landing pages, learn surfaces
PARTNER SURFACES

Operational. Restrained. Trustworthy.

Compact spacing, dense data tables, fewer animations, more information per pixel. The platform should feel like a tool partners reach for daily, not a marketing surface.

  • Order inbox · Verification queue · Production dashboards
  • Compact density (16–24px section spacing)
  • Inter only — no editorial serif
  • Optional dark mode default for long-session surfaces

§ 08

Dark mode

Not an inversion — a considered second palette. Ink 900 as canvas, terracotta desaturates one step (Terra 400) so it doesn't vibrate against the dark background. Default for long-session partner surfaces; opt-in for creator surfaces.

PRODUCTION QUEUE

You have 4 orders awaiting your review.

A calmer surface for the work-heavy hours. Surfaces step lighter (not darker) to create elevation. Borders use semi-transparent white, never solid gray. Type stays off-white, never pure white.

§ 11

Motion

Subtle and considered. 220ms ease-out for entering elements; 320ms ease-in for leaving. Animate transform and opacity only. No bouncy, no theatrical, no decorative loops. Hover this card.

Card hover

translateY(−3px) + shadow-lg over 220ms ease-out.

transform · 220ms · cubic-bezier(0.16, 1, 0.3, 1)

Button press

translateY(+1px) on :active, returns instantly on release.

120ms · ease-out

Page transition

Fade + 8px slide-up on route enter. Reverse on exit.

opacity · transform · 220ms

Every animation respects prefers-reduced-motion: reduce. Test by enabling reduce motion in your OS — animations disable, transitions stay instant. Try it.

Beyond contrast

  • Focus rings: 3px terra-500/15 ring on all interactive elements
  • Touch targets: 44 × 44px minimum (height-32 buttons have 8px hit-area padding)
  • prefers-reduced-motion: reduce respected globally
  • Color never used alone — icons and text always accompany state
  • Semantic HTML — <button>, <nav>, <main>, heading hierarchy
  • Keyboard navigation: Tab, Shift+Tab, Enter, Escape all wired

What to verify per surface

  • Every input has a visible label (placeholder-only is a fail)
  • Icon-only buttons have aria-label
  • Status changes have ARIA live-region announcements
  • Modal dialogs trap focus and return on close
  • Skip-link "Skip to main content" on every page
  • Run axe DevTools on every new page before shipping
